# Temporal Mechanics Configuration Service

Author: Robinson Collado

### To Run:
- From a terminal, enter and run `bun install` and `bun dev` in the root directory of the project. You should be able to install dependencies and run the project with npm, yarn, pnpm, etc. as well.
- Navigate to [http://localhost:5173](http://localhost:5173){:target="_blank" rel="noopener"} in your browser.
- Once on the home page, you need to store an api key in local storage so that the app can make api requests. Ex. `localStorage.setItem('x-api-key', '<YOUR-API-KEY>')`
