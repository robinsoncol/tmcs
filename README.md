# Temporal Mechanics Configuration Service

- Author: Robinson Collado
- To start the project, run `bun install` and then run `bun dev`. You should be able to install dependencies and run the project with npm, yarn, pnpm, etc.
- For the project to work, you need to store an api key into local storage in your browser. Ex. `localStorage.setItem('x-api-key', '<YOUR-API-KEY>')`
